视频批量重命名工具 v1.0
========================

【功能说明】
本工具用于批量重命名视频文件，支持以下格式：
MP4, AVI, MKV, MOV, WMV, FLV, WebM, M4V, MPG, MPEG, 3GP, TS, MTS, VOB

【使用步骤】

步骤1：扫描视频文件
--------------------
1. 点击"选择文件夹"按钮，选择包含视频文件的文件夹
2. 点击"扫描视频文件"按钮，程序会列出所有视频文件

步骤2：导出并编辑表格
--------------------
1. 点击"导出xlsx表格"按钮，保存xlsx文件
2. 使用 Excel 或 WPS 打开xlsx文件
3. 在"文件名（新）"列填写新的文件名（必须包含扩展名）
4. 保存xlsx文件
5. 点击"选择xlsx表格"按钮，加载编辑好的xlsx文件

步骤3：批量重命名
--------------------
1. 点击"预览重命名"按钮，检查重命名是否正确
2. 确认无误后，点击"执行批量重命名"按钮
3. 查看操作日志，确认重命名结果

【如何打包为EXE】
--------------------
方法1：双击运行 build_exe.bat
  - 会自动安装依赖并打包
  - 生成的EXE文件在 dist 文件夹中

方法2：手动打包
  pip install -r requirements.txt
  pyinstaller --onefile --windowed --name "视频批量重命名工具" video_rename_tool.py

【注意事项】
--------------------
1. 重命名操作不可撤销，请先用"预览重命名"检查
2. 新文件名必须包含扩展名（如 .mp4）
3. 新文件名不能与现有文件重名

【系统要求】
--------------------
- Windows 7/8/10/11
- Python 3.6+（如需运行源码）
- 依赖：openpyxl（用于读写Excel文件）
